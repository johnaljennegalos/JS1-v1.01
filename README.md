## How to Run Locally

1.  Clone the repository:
    ```bash
    git clone [https://github.com/aljnmusic/JS1-v1.01.git](https://github.com/aljnmusic/JS1-v1.01.git)
    ```
2.  Navigate to the project folder:
    ```bash
    cd JS1-v1.01
    ```
3.  Open the specific project folder (e.g., `/WeatherApp`) and launch `index.html`.
    * *Recommended:* Use the **Live Server** extension in VS Code for the best experience.
